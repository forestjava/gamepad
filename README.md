# Flight Control Demo

Приложение Flight Control Demo отображает в веб-клиенте положение стикеров джойстика управления, подключенного в системе, используя Gamepad API.

## Ветки

- `main` - сгенерированное LLM replit.com приложение
- `client` - чистое фронтенд react vite приложение

## План

- `module` - npm библиотека для разработки gamepad приложений

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Gamepad API

## Установка и запуск

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`
